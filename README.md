# Mern-Stack-Crud-App

This is a simple crud project for beginners built with the following technologies:
- [React] and [React Router]for the frontend
- [Express] and [Mongoose]for the backend


## Requirements

- [Node.js]

```shell
npm install
```


## Usage

- Backend
```shell
node server.js
```

- Frontend
```shell
cd client && npm start
```
